# WMI, Where am I?
> __무선 네트워크 기술을 활용한 사용자 인증 및 자원 통합 관리 플랫폼__ 입니다.  
> 부산대학교 김태운 교수님의 지도로 진행되며, 김태운 교수님의 자료를 참고합니다.

### 3. 시스템 구성도

본 과제의 가장 기본적인 목표는 아래와 같다.
